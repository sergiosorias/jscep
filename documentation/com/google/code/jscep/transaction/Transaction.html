com.google.code.jscep.transaction
Class Transaction

java.lang.Object
  com.google.code.jscep.transaction.Transaction

public class Transaction
extends Object

This class represents a SCEP transaction, and provides a framework for performing operations.

The behaviour of this class changes in accordance with the possible valid states for each transaction operation. For enrollment operations, clients should obtain the current state using getState(), and perform the steps required by that outcome. For all other operations, the method will return the required type unless the server rejects the operation, in which case a PkiOperationFailureException will be thrown.
