Class SignedDataGenerator

java.lang.Object
  com.google.code.jscep.pkcs7.SignedDataGenerator

public class SignedDataGenerator
extends Object

This class is used for generating degenerate (certificates and CRLs only) SignedData instances.

Example usage: 

 X509Certificate cert = ...;
 
 SignedDataGenerator gen = new SignedDataGenerator();
 gen.addCertificate(cert);
 SignedData signedData = gen.generate();
