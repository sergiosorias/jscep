org.jscep.transaction
Class Transaction

java.lang.Object
  org.jscep.transaction.Transaction

public class Transaction
extends Object

This class represents a SCEP transaction, and provides a framework for performing operations.

The behaviour of this class changes in accordance with the possible valid states for each transaction operation. For enrollment operations, clients should inspect the Transaction.State returned by the enrollCertificate(X509Certificate, KeyPair, char[]) method or the state returned by the callable returned by getTask(). For non-enrollment operations, either the invocation will be successful, or the method will throw a PKIOperationFailureException.
